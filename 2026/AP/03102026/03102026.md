# Modulo (resto)

10 % 3 = 1
12 % 3 = 1
5 % 7 = 5
0 % 5 = 0
30 % 4 = 2

# Operadores relacionales

- < menor que
- \> mayor que
- <= menor o igual que
- \>= mayor o igual que
- == igual que
- != distinto que

## Ejemplos

- 3 < 4 = true
- 3 > 8 = false
- 3 < 3 = false
- 3 <= 3 = true
- -10 > 7 = false
- 3 == 7 = false
- 3 != 5 = true
- true != true = false
- false == false = true

# Operadores lógicos

- && and
- || or
- ! not

# Tabla de la verdad

## AND

```ts
true && true = true;
true && false = false;
false && true = false;
false && false = false;
```

> [!NOTE]
>
> Los 2 valores tienen que ser `true` para que me den `true`, sino me da `false`.

## OR

```ts
true || true = true;
true || false = true;
false || true = true;
false || false = false;
```

> [!NOTE]
>
> Si alguno de los valores es `true`, el resultado también es `true`, sino es `false`.

## NOT

```ts
!true = false;
!false = true;
```

> [!NOTE]
>
> Cambia un `valor` por el `contrario`.

# Estructura condicional

## if

```java
if (condición) {
  // Instrucciones
}
```

> Si la condición es `true`, se ejecutan las instrucciones. Si es `false`, no se ejecutan.

## if else

```java
if (condición) {
  // Instrucciones
} else {
  // otras instrucciones
}
```

> Si la condición es `true` ejecuta las instrucciones del primer bloque. Sino ejecuta las del segundo `else`.
