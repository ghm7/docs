> [!NOTE]
> SO -> Software base
> **Definición:** Es un software especializado en la administración del hardware de una computadora. Provee la base para que las aplicaciones puedan ejecutar y asi mismo es el intermediario entre el usuario, los programas y el hardware.

**Windows** -> Privado

**Linux** -> Libre

**Mac OS** -> Híbrido: tiene capa privada y capa libre, el core y lo mas interno es libre mientras que la UI es privada, es lo que Apple vende.
