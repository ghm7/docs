El sistema operativo GNU/Linux tiene una historia que nace de dos proyectos, por un lado GNU iniciado por Richard Stallman en 1983. Por otro lado el estudiante fines Linus Torvalds. En 1991 crea el software de base para una tesis de grado. \
Richard Stallman unifica los dos proyectos y nace el sistema operativo que conocemos con el nombre de GNU/Linux. \
Muchas personas de todas partes del mundo colaboraron en el desarrollo de nuevas versiones. \
GNU/Linux es un sistema operativo **LIBRE**.

Que significa libre?

Antes de empezar a definir que es 'libre' es necesario separarlo del concepto de free, esto se fundamenta en los significados que se producen en la lengua inglesa sobre una misma palabra. Free puede entenderse entre otras cosas como 'libre', pero también como 'gratis'.

Tiene que tener 4 características

- Ley 0: Uso sin limites.
- Ley 1: Tenes acceso al código.
- Ley 2: Libertad de distribución.
- Ley 3: Todas las modificaciones tienen derecho de publicarlo.

# Que es una licencia de software?

Una licencia de software especifica los derechos y obligaciones del uso para un determinado software o su código fuente. \
Las licencias pueden agruparse por diferentes categorías y entre las mas comunes se encuentran las que tienen que ver con el grado de libertad respecto al uso, distribución y modificación de un programa informático o su código fuente.

a- Licencias privadas \
b- Licencias permisivas \
c- Copy Left \
d- Weak Copy Left \

- **a:** esta licencia no permite acceder al código fuente. Aquí solo adquirimos el derecho de ejecutarlo, pero no a conocer como se hizo su implementación.

- Windows y Office de Microsoft.
- Adobe Photoshop.

b, c y d - es un grupo de licencias donde se modela el software libre. Cada una de ellas agrupa derechos y obligaciones que la diferencian entre si.

- **b:** MIT o APACHE - son dos ejemplos de licencias permisivas, aquí no aparecen restricciones sobre código, por ejemplo, puedo cambiar su licencia. Puedo tomar un producto y patentarlo privativamente.
- **c:** Copy Left es mas restrictivo, con el fin de proteger el código abierto de un producto. GPL2 y GPL3 son las mas restrictivas, se puede modificar, publicar y acceder al código fuente pero siempre bajo esta licencia. Ejemplo:
- GNU/Linux.
- Libre Office.
